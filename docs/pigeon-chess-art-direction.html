Art Direction Vision

🎨 The One Sentence Pigeon Chess looks like a prestigious Victorian chess journal that has been slowly taken over by pigeons and refuses to acknowledge it.

The visual identity of Pigeon Chess is built on a single tension: dignified formality vs. escalating chaos. The base aesthetic is serious, classical, and slightly dusty — aged parchment, engraved typography, candlelit board warmth. This is the world Viktor believes he still inhabits. Over this foundation, chaos accumulates: mold spreads, tiles catch fire, modifier cards reference internet memes, a pigeon named Gerald sits on your queen. The board does not explain itself.

The art direction must always serve both sides of this tension simultaneously. A lava tile should feel genuinely dramatic — not cartoonish. Gerald's arrival should feel genuinely absurd — not whimsical. The deadpan is the joke. The gravitas is what makes the pigeons funny.

Visual Pillars

Pillar 1 — Deadpan Grandeur Every game element is presented with maximum seriousness regardless of content. Lava tiles have a formal Latin label. Gerald gets a nameplate. The "YOU DIED" screen uses a classical serif font. The absurdity lives in the content, never the presentation style.
Pillar 2 — Organic Corruption The visual language starts clean and becomes progressively corrupted. Fresh board: pristine squares, warm light, crisp pieces. Madness at 75%: board grain increases, ink bleeds at tile edges, pieces cast wrong shadows. Full Flock: everything feathers.
Pillar 3 — Tactile Physicality Everything feels like it exists in a physical world. The board has weight. Pieces have wood grain. Cards have paper texture. No flat design. No gradients for their own sake. Light sources are consistent. Shadows are real.
Pillar 4 — Modifier Visual Identity Each of the five modifier categories has a distinct visual language — not just a colour, but a texture, a border treatment, a particle effect. Players should be able to identify a modifier category at a glance from across the screen.

Colour Rules

⛔ Colour Prohibitions 1. No pure white (#ffffff) anywhere. Use Parchment (#f2ead8) or Cream (#faf7f0) instead. Pure white reads as digital and breaks the physical world illusion.
2. No cyan or teal in the base palette. These are reserved for Fog of War effects only — seeing cyan elsewhere is a visual signal that something is hidden.
3. No flat gradient backgrounds on game surfaces. Texture overlays required on all large background areas.
4. Category colours are exclusive. A board mutation tile must not use Category C (ability) blue. Visual language must be consistent.

Typography

Three typefaces. No substitutions. Each serves a defined role and must not be used outside it.

Display — Playfair Display · Headings, Titles, Viktor's Dialogue, Modifier Names
Pigeon Chess
"The board has opinions."
Playfair Display — Usage Rules Use 900 weight for all game titles, section headers, and major UI labels. Use 400 italic for Viktor's dialogue lines, modifier flavor text, and pull quotes. The contrast between the two weights IS the visual voice of the game — serious headline, wry italic aside. Never use Playfair Display in all-caps. Never use it at sizes below 14px.
Body — DM Sans · All UI text, descriptions, rules, tutorial copy
Viktor Crumb placed his knight on d4. The knight did not ask for this. The knight has an active ability cooldown of 6 turns and a deeply complicated relationship with the concept of legal movement. Viktor is aware of this. Viktor does not care.
DM Sans — Usage Rules 300 weight for all body copy, descriptions, and paragraph text. 500 weight for UI labels, button text, and emphasized information. Never bold (700) — use 500 as the heavy weight. DM Sans is the utilitarian counterpart to Playfair's expressiveness. It should feel invisible — clear, never decorative.
Mono — DM Mono · IDs, Stats, Timers, Code, System Labels
MOD-B002 · CURSE · TURN 14 · COOLDOWN 3 · ELO +24 · FEATHERS ×7
DM Mono — Usage Rules Used exclusively for data, identifiers, timers, ELO numbers, modifier stats, and system labels. Letter-spacing: 0.1em minimum. Color: always a secondary/dim tone — never primary text color. DM Mono signals "this is a number, a code, a system value" — it should never appear in descriptive prose.

Board Design

Default Board — "The Crumbed Board"

The default board is Viktor's cursed board — aged walnut wood, warm candlelit light, slight grain. It should feel like a physical object from a high-end Victorian chess set that has seen better days.

Dark Square Spec

  • Base colour: #1e1408 (Board Dark)
  • Wood grain texture: subtle horizontal lines, 2px spacing, 3% opacity
  • Ambient light: slight top-left warmth (radial gradient 8% opacity)
  • Border: 1px inset shadow, 15% darker than base

Light Square Spec

  • Base colour: #c8a870 (Board Light)
  • Wood grain texture: same as dark, but lighter grain
  • Ambient light: slightly more luminous at centre
  • Border: 1px inset shadow matching dark square treatment

Board Frame Spec

  • Outer frame: 12px, deep walnut #100a04
  • Rank/file labels: DM Mono, 10px, #8a6840
  • Corner rosettes: small carved pigeon motif (4px × 4px)
  • Drop shadow: 8px offset, 40% opacity, warm tone

Selected Piece Highlight

  • Selected square: #c8973a at 40% opacity overlay
  • Legal move dot: 12px circle, #c8973a at 60% opacity
  • Capture target: full square red tint #b83228 at 30%
  • King in check: full square red tint pulsing 0.5s ease

Board Coordinate System (Phaser)

The board renders in a Phaser Scene at a base resolution of 640×640px (80px per square). The scene scales to fill available container space using Phaser's ScaleManager with FIT mode. All tile effect overlays are rendered as separate Phaser layers above the board base layer, allowing independent opacity and blending control per effect type.

Piece Design

All pieces in the default set are rendered as top-down 2D illustrated sprites at 64×64px base resolution (scaled to 72×72 on the 80px board with 4px padding). The art style is engraved illustration — high-contrast, slightly aged, as if from a Victorian chess manual. No 3D. No gradients. Strong outlines with internal hatching for depth.

🎨 Style Reference Target look: Staunton chess pieces as if drawn by a 19th-century engraver for a natural history journal. Think Audubon bird illustrations but for chess pieces. Slightly absurd anatomical accuracy. Every piece takes itself completely seriously.

Gerald has seen things. Gerald is not impressed by any of this.

Tile Effects

Every Board Mutation modifier produces a distinct tile visual state. These must be instantly recognisable, visually distinct from each other, and consistent with the "physical world" aesthetic — each effect looks like something that could genuinely happen to a wooden chess board.

Board Demo — Active Tile Effects

🔥 Lava (The Floor Is Lava)
MOD-A002 — Persistent
Glowing orange-red tile with animated heat shimmer. Small flame particles emitting upward at 30% opacity. Tile border glows amber. Rotation preview shown as amber dashed outline on adjacent target square.
Base: #8a2020 · Glow: #e85020 · Particles: 4fps · Shimmer: 2s sine wave
🍄 Infected (The Cordyceptic Spread)
MOD-A001 — Spreading
Dark green tile with mycelium texture overlay — fine white thread network drawn as static illustration. Slow pulsing glow at 4s intervals. Adjacent uninfected tiles show a faint green tint as "spread warning" when infection is 1 turn from spreading.
Base: #1a2808 · Mycelium: #304818 · Pulse: 4s ease-in-out · Spread warning: 20% opacity
❄️ Frozen (Winter Is Coming)
MOD-A004 — Zone
Blue-grey tile with ice crack texture overlay. Small snowflake particles drifting downward very slowly. A piece on a frozen tile shows a frost ring around its base. Thawing piece has the frost ring slowly dissolving over the turn.
Base: #182838 · Cracks: #304858 at 60% opacity · Particles: 8fps, slow drift · Frost ring: 2s dissolve
🌿 Swamp (This Is My Swamp)
MOD-A006 — Zone
Dark green-brown tile with occasional bubble animation at the surface. Pieces in the swamp have a slow "sinking" idle animation (bob 2px down). The swamp makes a squelching sound on entry. It does not apologise for this.
Base: #182010 · Bubbles: 6s random interval · Piece bob: 2px, 3s sine · Border: muddy edge texture
⬛ Void (Storm Consumed)
MOD-A005 — Storm
Complete black with faint edge crumble particles falling inward. Not fog — void. The tile is gone. A piece eliminated by the storm produces a small dust effect before disappearing. No pieces can enter void squares.
Base: #040404 · Edge particles: inward-falling, 2px dust, 3fps · Piece elimination: 0.3s dust burst
🌑 Fog (It's Dark And I'm Scared)
MOD-D001 — Fog
Near-black with a subtle deep blue undertone. The boundary between visible and fog is not hard-edged — a 2-square gradient feathering at the visibility edge. This makes the board feel like it exists in darkness rather than being cut off.
Base: #080810 · Boundary: 2-square gradient feather · Cat Mode: boundary shifts 1 square outward
👁️ Surveillance (Big Beak Is Watching)
MOD-D008 — Surveillance
Normal tile with a golden eye symbol rendered at the centre at 50% opacity. When a piece enters the 1-square radius, the eye briefly opens fully (100% opacity, 0.3s) and the tile pulses gold. The eye closes again slowly. Very unsettling.
Eye: Playfair Display "👁" glyph, 24px · Open pulse: 0.3s ease · Ambient: slow blink, 8s interval
⚡ No-Fly Zone (I Hereby Declare)
MOD-C001 — Active
A golden line across the entire row or column. Not a tile effect — a Phaser Graphics overlay at depth 70. Line is 2px, dashed, animated dash-offset. Small crown icon at each end of the line. Opponent pieces attempting to cross are visually blocked and snapped back.
Line: 2px dashed, #c8973a · Dash animation: 0.5s scroll · Crown icons: 16px at line ends

Ability Button

When a piece with an active ability is selected, an ability button appears adjacent to the board at the piece's edge. It shows: ability icon, name (abbreviated), cooldown remaining (in turns). The button pulses with the category colour when the ability is ready. When on cooldown, a circular progress ring depletes around the button. Using the ability consumes the player's turn — a confirmation step appears for high-cost abilities (Quiet Quitting, Equivalent Exchange).

Animation Direction

🎬 Animation Philosophy Animations in Pigeon Chess serve the deadpan tone. Chaos events are presented with the visual gravity of important chess moments. Gerald lands with the same ceremonial weight as a checkmate. The "YOU DIED" screen appears with operatic slowness. Nothing winks at the player. The game takes all of it completely seriously.

Board Animations

Piece Move
On every legal move
Piece lifts slightly (4px scale-up, shadow appears), slides to destination along a slight arc, lands with a subtle bounce. Duration: 180ms. Pieces do not slide through other pieces — they arc above.
Duration: 180ms · Easing: cubic-bezier(0.34, 1.56, 0.64, 1) · Arc height: 8px
Capture
On piece capture
Capturing piece moves normally. Captured piece does a brief shake (0.1s) then fades out with a small puff of board dust particles. Not cartoonish — dignified. The piece accepts its fate with composure.
Shake: ±3px, 3 oscillations, 100ms · Fade: 200ms · Dust: 8 particles, 300ms
Gerald Lands
MOD-B002 turn start
Gerald drops from above with a slight wing-flutter. Landing produces a small ripple on the tile. Gerald's nameplate slides in from the left (0.2s). The targeted piece's available moves dim. Gerald settles and looks around, unimpressed.
Drop: 200ms, ease-out · Nameplate slide: 200ms · Wing flutter: 3-frame sprite
Rook Cannon Fire
MOD-C006 activation
Rook shudders briefly. A cannonball sprite (small, dark sphere with smoke trail) travels the board in 0.4s. It hits its target with a flash and a brief screen shake (2px, 100ms). Captured piece explodes into dust fragments.
Travel: 400ms linear · Screen shake: ±2px, 100ms · Impact flash: #ffaa00, 150ms fade
YOU DIED Screen
MOD-B001 autonomous move
Screen fades to near-black over 300ms. Large red text "A PIECE MOVED" fades in over 500ms in Playfair Display 900 weight. Holds for 1200ms. Fades out. Control returns. The piece looks pleased. The player does not.
Fade in: 500ms · Hold: 1200ms · Fade out: 300ms · Total: 2000ms
Gravity Flip
MOD-A007 every 12 turns
The entire board tilts 15 degrees in the new gravity direction (0.6s), all pieces slide simultaneously, the board returns to flat (0.4s). Pieces sliding off the edge don't fall — they dissolve at the boundary. Total: 2s. Phaser camera tween controls the board rotation.
Tilt: 15deg, 600ms ease-in-out · Slide: 400ms · Return: 400ms ease-out
Betrayal / Defection
MOD-B006 trigger
Defecting piece glows red for 1 full turn (warning). On defection: piece flips colour (colour-swap sprite tween, 400ms), Among Us-style ejection card appears at screen bottom for 1.5s: "[Piece] was The Traitor. [N] pieces remain." Card slides in from bottom, pauses, slides out.
Warning glow: 1 full turn · Colour flip: 400ms · Card slide: 300ms in/out · Hold: 1500ms
BOY. Chain Capture
MOD-E006 each chain step
Each chain capture plays the standard capture animation at 0.7x speed. A golden trail follows the Berserker piece between captures. After the chain ends, the Berserker piece does a brief "victory bob" (2px up/down, 200ms). If 3+ captures in chain, screen edge flashes amber once.
Each capture: 126ms (0.7x) · Trail: golden line, fades 600ms · Victory bob: 200ms · 3+ flash: 300ms

Draft phase opens

Viktor & Characters

Viktor Crumb — Visual Design

👤 Viktor Crumb — Character Brief Viktor is a disgraced chess grandmaster in his early 50s. He dresses as if still competing at the highest level — formal, slightly too formal for wherever he currently is. His expression is permanently composed — the expression of a man who has decided that he will not be surprised by anything ever again, and is being regularly surprised. His posture is immaculate. Everything else is going wrong.

Portrait Spec

  • Format: Illustrated portrait, 200×260px, waist-up. Same Victorian engraving style as pieces — hatching, high contrast, aged warmth.
  • Base expression: Composed. Controlled. A man maintaining dignity under pressure.
  • Expression variants: 5 core states — Composed (default), Unsettled (Madness 26–50), Unravelling (51–75), Accepted (76–99), Full Flock (100, eyes wide, pigeon on head).
  • Clothing: Dark formal jacket, white shirt, chess tournament badge still pinned from his last official match (11 years ago). The badge is slightly askew by Madness tier 3.
  • Pigeon accumulation: At Madness 51+, a pigeon appears on Viktor's shoulder in the portrait. At 76+, two pigeons. At Full Flock: Viktor is covered in pigeons. The portrait still shows his face. Viktor's expression remains composed.

Opponent Characters (Story Mode Bosses)

TowerBossVisual Direction

Reginald is a pigeon wearing a small monocle and a press badge that reads "CRITIC" in DM Mono. His review panel is 200×60px. Portrait thumbnail: 40×40px, circular crop. He has a resting expression of profound disappointment. When delivering a high score (8/10+), his expression does not change. The score alone communicates his surprise at being impressed.

Madness Visual States

The Madness Meter (0–100, derived from active curse ratings) progressively corrupts the visual environment. This is the most technically complex part of the art direction — it requires coordinated changes across Phaser (board layer), React (UI chrome), and audio. Cosmetics Design System

Cosmetics replace visual elements without changing gameplay. Every cosmetic must respect the deadpan grandeur aesthetic while allowing creative variation. Community-created cosmetics go through an art review process to ensure consistency.

Cosmetic Categories

CategoryWhat It ReplacesConstraintsExample Themes
Board Themes Full board visual — tile colours, frame design, grain, ambient light Must maintain light/dark square legibility at all Madness states. No tiles that make piece identification ambiguous. Marble Palace, Moonlit Garden, Space Station, Coral Reef, The Sewers (Shrek themed)
Piece Sets All 6 piece types (or subset). Can be full sets or partial (pawns only, etc.) Must maintain size proportions. King must always be tallest. Community Cosmetic Submission Guidelines

📋 Art Review Checklist (All Community Submissions) 1. Style consistency: Must match Victorian engraving aesthetic OR commit to an alternate style that is complete and self-contained (not a mix of styles within one cosmetic).
2. Legibility at all sizes: Test at 50% zoom. Pieces must still be distinguishable from each other.
3. Madness compatibility: The Madness tint system overlays on all board themes. Verify the theme still reads correctly at Full Flock plum tint.
4. Category stripe preservation: Modifier card art must not interfere with the top category stripe colour.
5. No real trademarked characters or likenesses. Pop-culture adjacent is fine. Direct copies are not.
6. File formats: PNG for static sprites, spritesheet PNG for animated. Max 512×512 per sprite. Spritesheets: max 2048×2048.
